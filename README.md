Data Analytics Project
Overview

This project demonstrates a complete Data Analytics workflow using Python, SQL, and Power BI. The project involves loading and cleaning datasets, performing Exploratory Data Analysis (EDA), executing SQL queries in relational databases, building interactive dashboards, and presenting insights through reports and presentations.

The main objective of this project is to extract meaningful business insights from raw data and visualize them in an easy-to-understand format.

Dataset

The dataset used in this project contains structured data related to business operations, sales, customers, or other analytical domains.

Dataset Features
Multiple columns with numerical and categorical values
Missing and duplicate data handling
Data transformation and preprocessing
KPI and trend analysis
